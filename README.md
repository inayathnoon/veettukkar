# Veettukkar — Kerala's Home Services Marketplace

A Malayalam-first mobile app connecting Kerala homeowners with verified local daily-wage workers — coconut tree climbers, house painters, cleaners, cement construction workers, plumbers, and electricians. Homeowners post a job; nearby workers are notified and accept. Payment stays in cash. The app handles matching, trust, and reliability.

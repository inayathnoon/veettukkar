# Decision: MVP Scope — Veettukkar

## What's In and Why

The MVP scope proves one thing: **can the platform reliably connect a homeowner with a nearby daily-wage worker in Ernakulam district?**

### What's Deliberately Excluded from v1

| Excluded Feature | Why Excluded |
|-----------------|--------------|
| iOS app | ~95% of target demographic is Android |
| In-app payments | KYC + regulatory overhead; cash model works |
| Multi-district | Cold-start must be solved in one district first |
| Job categories beyond 6 | Validate model before expanding supply |
| Background checks beyond Aadhaar | Too slow and expensive for v1 scale |
| Worker-to-homeowner public ratings | Added in v2 after trust established |
| AI/ML matching | Proximity + rating ranking is sufficient |
| Admin dashboard | Solo founder handles moderation manually |
| Subscription plans | Revenue model deferred to v2 |

## Geographic Scope

**Launch:** Ernakulam district only.

**Why Ernakulam:**
- Highest construction activity in Kerala
- Dense migrant worker population (Aluva, Perumbavoor, Edapally)
- Urban Company's only strong Kerala foothold — can undercut on worker-friendliness
- High digital literacy (Kochi metro)

**Expansion sequence (post-PMF):**
1. Ernakulam ✓
2. Thiruvananthapuram
3. Kozhikode
4. Thrissur
5. Remaining 10 districts

Each new district requires 20+ verified workers per skill category before opening to homeowners.

## Cold-Start Strategy

Before the app goes live publicly:

1. **In-person recruitment:** Visit 3–5 construction sites in Aluva/Perumbavoor with printed registration cards + WhatsApp onboarding flow
2. **Kudumbashree network:** Kerala's 300,000-member women's self-help groups — reach via district coordinators for cleaners
3. **Hello Nariyal directory:** Coconut Development Board's 1,924 registered climbers — contact directly
4. **Minimum threshold:** 30+ verified workers across all 6 categories before first homeowner onboarding begins
